# Tridiv Swain — Portfolio

Personal portfolio website showcasing research publications, work experience, and skills.

## Live Demo

Deploy via GitHub Pages: **Settings → Pages → Source: main branch → / (root)**

Your site will be live at: `https://<your-username>.github.io/tridiv-portfolio/`

## Tech Stack

- Pure HTML, CSS & JavaScript — zero dependencies
- Responsive design (mobile, tablet, desktop)
- Scroll-reveal animations via IntersectionObserver
- Split-screen hero inspired by modern portfolio design

## Features

- **Split-screen hero** — Data Scientist / ML Engineer duality
- **10 published research papers** with project cards and images
- **Work experience timeline** — PwC, HighRadius
- **Education section** — NUS, KIIT, Higher Secondary
- **Contact section** with social links
- **Smooth scroll navigation** with glassmorphism navbar

## Deployment

### GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main** branch, folder **/ (root)**
4. Your site will be live in ~1 minute

### Netlify / Vercel
Simply drag-and-drop the folder or connect your GitHub repo.
